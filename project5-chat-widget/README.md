# AI Customer Service Chat Widget

An embeddable AI chat widget that answers customer questions 24/7 and helps local businesses convert more website visitors into leads.

## Problem It Solves

Many businesses lose opportunities because website visitors have questions outside business hours and leave before anyone can respond.

## Target Client

Local businesses such as dental offices, salons, HVAC companies, and other service providers with repetitive website inquiries.

## Tech Stack

- HTML
- CSS
- JavaScript
- OpenAI GPT-4o-mini API

## Configuration and Deployment

1. Update the widget configuration with the client's business name, hours, phone number, services, and support prompt.
2. Replace the demo API setup with a secure backend proxy for production use.
3. Customize the branding, welcome message, and fallback behavior for the client's site.
4. Deploy the widget on a static host or embed it directly into the client's website.
5. Test common support questions and escalation scenarios before launch.

## Demo Instructions

1. Open the widget and ask realistic customer questions such as `What are your hours?` or `Do you offer whitening?`
2. Watch the chatbot respond instantly in a friendly, business-specific tone.
3. Show how the widget keeps working after hours when nobody is available to answer manually.
4. Explain that the business can now handle common questions 24/7 without losing visitors who were ready to book.
