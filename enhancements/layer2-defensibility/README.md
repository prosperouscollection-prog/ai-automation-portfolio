# Layer 2: Defensibility Assets

Layer 2 turns Genesis AI Systems from a collection of deliverables into a defensible operating system. These assets strengthen pricing power, increase retention, and make replacement harder over time.

## Included Assets

- `PROMPT_LIBRARY.md`: proprietary prompt IP by use case and vertical
- `PROMPT_VERSIONING.md`: operating playbook for prompt changes and A/B tests
- `training_data/`: vertical-specific fine-tuning starter datasets
- `TRAINING_DATA_README.md`: how to grow client datasets into custom models
- `CLIENT_DATA_MOAT.md`: strategy for building switching costs through accumulated operational data
- `referral_system.py`: a referral engine that tracks sources, commissions, and conversions
- `REFERRAL_PROGRAM.md`: client-facing referral offer sheet
- `referral_email_templates.md`: ready-to-send referral email copy

## Why This Layer Matters

- makes your systems look proprietary instead of generic
- gives clients a reason to stay longer because the system gets better over time
- creates assets you can reuse across multiple clients and verticals
- raises the ceiling on custom setup fees and strategic retainers
