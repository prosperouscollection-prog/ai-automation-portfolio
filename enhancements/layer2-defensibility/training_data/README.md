# Training Data Templates

This subfolder contains vertical starter datasets for Genesis AI Systems fine-tuning projects.

Files included:

- `dental_training_data.jsonl`
- `hvac_training_data.jsonl`
- `salon_training_data.jsonl`
- `restaurant_training_data.jsonl`
- `real_estate_training_data.jsonl`

Each template uses `25` examples to make the first fine-tuning engagement faster and more repeatable. Expand these with real client interactions after deployment.
