# Language Support Strategy

## Year 1 Languages

- English
- Spanish

## Why Spanish

There are roughly `41 million` Spanish speakers in the United States, and they remain significantly underserved by many automation vendors.

## Opportunity

Genesis AI Systems can charge `+25%` for bilingual systems because they:

- expand the client’s reachable market
- improve customer experience
- reduce missed leads caused by language mismatch

## How to Build Bilingual Voice Agents

- create English and Spanish greeting branches
- detect language early in the call
- route into language-specific prompt instructions
- keep escalation and safety rules consistent across languages

## How to Build Bilingual Chatbots

- detect language from the first user message
- route into the correct prompt set
- allow users to switch languages mid-conversation when needed

## How to Build Bilingual Lead Capture Systems

- store original message language
- score intent consistently across languages
- use bilingual auto-responses when confidence is high
