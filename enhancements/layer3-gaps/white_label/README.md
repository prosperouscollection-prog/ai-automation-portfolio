# White Label Package

This subfolder contains the reseller-facing white label system for Genesis AI Systems.

Included assets:

- `WHITE_LABEL_GUIDE.md`
- `RESELLER_AGREEMENT_TEMPLATE.md`
- `REBRANDING_CHECKLIST.md`

Use this package when an agency wants to resell Genesis AI Systems solutions under its own brand while Genesis AI Systems remains the backend fulfillment partner.
