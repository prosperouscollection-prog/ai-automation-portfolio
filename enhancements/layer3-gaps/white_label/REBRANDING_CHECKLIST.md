# Rebranding Checklist

## Goal

Rebrand a Genesis AI Systems deployment for a reseller in roughly `2-3 hours`.

## Update These Elements

- colors
- logos
- email sender name
- support contact info
- dashboard labels
- chatbot greeting
- voice agent name where approved

## Files or Variables to Review

- HTML branding constants
- environment variables
- report headers and footers
- email templates
- prompt business name variables

## Commercial Note

Charge a `+$500` rebranding fee per system to cover setup and QA.
