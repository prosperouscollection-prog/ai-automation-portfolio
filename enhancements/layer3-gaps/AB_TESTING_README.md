# A/B Testing Framework

## Overview

`ab_testing.py` helps Genesis AI Systems improve prompts with evidence instead of guesswork.

It:

- splits traffic between Variant A and Variant B
- records quality and follow-up outcomes
- calculates statistical significance
- promotes a winning prompt when the data supports it

## When to Use It

- lead scoring prompt improvements
- auto-response prompt tests
- escalation language experiments
- booking CTA testing

## Important Rule

Only test one meaningful variable at a time. If you change everything at once, you learn nothing.
