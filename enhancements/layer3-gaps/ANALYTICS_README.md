# Usage Analytics

## Overview

`usage_analytics.py` helps Genesis AI Systems understand actual cost, margin pressure, and overages per client.

This is how you prevent a profitable client on paper from becoming an unprofitable client in practice.

## What It Tracks

- API calls
- tokens used
- leads processed
- voice minutes
- emails sent

## What It Produces

- estimated variable cost
- overage amount
- invoice addendum text
- profitability visibility by client

## Recommended Use

- record usage daily or per workflow run
- review summaries before invoicing
- watch for clients whose usage patterns drift outside their plan assumptions
