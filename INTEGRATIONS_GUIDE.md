# Third Party Integrations — Genesis AI Systems
# Tools we use instead of building from scratch

## Apollo.io — Find new leads

Free tier: 50 prospect exports per month  
Paid: $49/month

Setup:

1. Sign up with info@genesisai.systems
2. Open settings
3. Create an API key
4. Add it to GitHub secrets
5. Turn on the lead finder

## HubSpot — Keep leads organized

Free forever for core features

Setup:

1. Sign up free
2. Create a private app
3. Turn on contact write access
4. Copy the token
5. Add it to GitHub secrets
6. Add it to Render too

## Lindy AI — Sales follow-up

Free tier: 40 tasks/month  
Paid: $49/month

Setup:

1. Sign up
2. Create 3 follow-up paths
3. Hot
4. Warm
5. Cold
6. Copy the API key
7. Add it to GitHub secrets

If Lindy is not ready yet, the sales helper falls back to direct email.
