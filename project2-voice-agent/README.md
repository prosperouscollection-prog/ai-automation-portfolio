# AI Voice Agent

A 24/7 AI phone agent that answers inbound calls, handles common questions, and captures caller details for follow-up or booking.

## Problem It Solves

When phones go unanswered after hours or during busy times, service businesses lose high-intent leads they already paid to attract.

## Target Client

Dental offices, salons, HVAC companies, and real estate agents.

## Tech Stack

- Vapi
- OpenAI GPT-4
- ElevenLabs voice

## Configuration and Deployment

1. Create the voice agent inside Vapi and connect it to the client's phone number or call-forwarding setup.
2. Configure the business prompt with services, hours, FAQs, and lead capture questions.
3. Use OpenAI GPT-4 to handle call logic and response generation.
4. Assign an ElevenLabs voice that fits the client's brand and customer experience.
5. Test after-hours calls, urgent scenarios, and data capture before deployment.

## Demo Instructions

1. Call the Vapi number connected to the agent.
2. Say, `I have a toothache` to simulate a real high-intent caller.
3. Listen as Riley responds naturally, handles the situation professionally, and starts collecting the caller's information.
4. Explain that the business can now capture urgent leads even when nobody is available to answer live.
